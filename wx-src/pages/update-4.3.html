Panic Button has been upgraded to version 4.3

What’s New

The default keyboard shortcut for triggering the Hide All Windows action has changed. It is now ALT+F9.

Why did it change? Because F9 toggles Reader Mode in Firefox 78 and newer, so that key couldn’t be used anymore.

The keyboard shortcut can be changed by going to Manage Extension Shortcuts in Add-ons Manager.
